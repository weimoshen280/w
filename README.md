# AI智能招聘网站

一个完整的AI招聘系统，包含求职者端和HR管理端，集成了AI简历解析、智能人岗匹配、AI面试等核心功能。

## 项目结构

```
ai-recruitment/
├── login.html                    # 身份选择页面
├── css/
│   └── styles.css               # 全局样式
├── js/
│   └── app.js                   # 全局JavaScript（AI功能、数据管理）
├── candidate/                   # 求职者端
│   ├── index.html              # 首页
│   ├── jobs.html               # 职位搜索
│   ├── resume.html             # 简历管理
│   ├── applications.html       # 投递记录
│   └── interview.html          # AI面试
└── hr/                         # HR管理端
    ├── index.html              # 工作台
    ├── jobs.html               # 职位管理
    ├── resumes.html            # 简历筛选
    ├── interviews.html         # 面试管理
    └── talent.html             # 人才库
```

## 核心功能

### 1. AI智能功能
- **AI简历解析** - 自动识别简历关键信息，提取技能、工作经历、教育背景
- **智能人岗匹配** - 基于算法分析简历与职位匹配度，精准推荐合适岗位
- **AI视频面试** - 智能面试问题生成，实时评分分析，生成面试报告
- **AI简历优化建议** - 语法润色、关键词补齐、亮点提炼

### 2. 求职者端
- 手机号注册/登录
- 简历在线编辑与上传
- 职位搜索与筛选
- 一键投递
- 投递进度追踪
- AI模拟面试

### 3. HR管理端
- 企业信息管理
- 职位发布与编辑
- AI简历筛选
- 面试管理
- 人才库管理
- 数据看板

## 技术特点

1. **响应式设计** - 适配PC、平板等设备
2. **现代CSS** - 使用Flexbox和Grid布局，实现美观的视觉效果
3. **JavaScript交互** - 实现职位筛选、表单验证、平滑滚动等功能
4. **AI功能模拟** - 模拟AI简历解析、人岗匹配、面试评分等功能
5. **数据存储** - 使用LocalStorage在浏览器中存储数据，无需数据库

## 后续扩展

- **后端服务** - 开发Node.js后端，使用MongoDB存储数据
- **AI集成** - 集成百度或阿里云的AI API，实现真实的AI功能

- **扩展功能** - 添加付费、内推、背调等增值功能

## 注意事项

- 本系统为前端演示版本，使用LocalStorage存储数据
- 刷新浏览器页面后，数据会保持但可能需要重新登录
- AI功能为模拟实现，实际效果需要集成真实的AI服务
